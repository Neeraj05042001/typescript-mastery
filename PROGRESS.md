## Course quality standard

A lesson is treated as fully complete only when it has:

- A clear learning outcome and place in the curriculum
- Focused notes with realistic examples
- Exercises that require independent thinking
- Worked solutions that explain the reasoning
- Navigation to the previous and next learning step
- Compilable solution code where applicable

Update this page whenever a lesson or project is published. Accurate progress is part of the learning experience and part of the repository's credibility.
