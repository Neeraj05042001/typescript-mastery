# Changelog

All notable learner-facing changes to TypeScript Mastery are documented here.

The format is inspired by [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), adapted for an educational repository.

## Unreleased

### Added

- Root learning navigation: README, Start Here, Roadmap, and Progress pages.
- A public 33-lesson curriculum roadmap.
- Open-source contribution and community guidance.
- Repository quality checks for solution code.

### Changed

- The repository is being standardized around clear learner paths, honest progress, and phase-based milestones.

## How to maintain this file

Add learner-visible changes under **Added**, **Changed**, **Fixed**, or **Removed**. Group related updates into a dated release when a phase, assessment, or milestone project is ready to share.
