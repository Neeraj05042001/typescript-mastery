# Phase 1 — Foundations

> **7 lessons · Available now**

Build the TypeScript base that makes every later phase easier: how the compiler works, how values are typed, and how to describe the data and functions in a real application.

## You will be able to

- Set up a TypeScript project and understand `tsconfig` choices.
- Use primitive, array, object, and function types with confidence.
- Model object contracts with type aliases and interfaces.
- Read common compiler errors without immediately reaching for `any`.

## Completion checkpoint

Before moving on, you should be able to configure a strict TypeScript project and describe a small application domain with clear object types, functions, aliases, and interfaces.

The Phase 1 assessment is being prepared. Until then, revisit the exercises and explain each concept without looking at your notes.
